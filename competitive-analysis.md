## 各竞品分析

### MOODA — 最直接的竞品
- **定位**: 可爱治愈系情绪日记，韩系画风
- **价格**: 免费 + 会员 ¥8/月 ¥58/年
- **核心**: 情绪记录 + 房间装饰 + 好友互动
- **优势**: UI可爱，有社区氛围，免费版功能够用
- **弱点**: 情绪记录偏表面（选表情而已），没有"行动引导"
- **对我们的启示**: 可以做"情绪+行动"的差异化，MOODA只记情绪，我们推动用户做治愈小事

### 小日常 — 习惯打卡工具
- **定位**: 极简习惯打卡，零社交无广告
- **价格**: 免费 + VIP订阅
- **核心**: 打卡 + 番茄钟 + 数据统计
- **优势**: 干净纯粹，150万+下载，打卡仪式感强
- **弱点**: 没有情绪维度，纯工具属性，用户粘性靠自律
- **对我们的启示**: 我们的"抽取小任务"比"打卡"更有趣，但缺少小日常的"完成感"（成就卡片）

### 心光 — AI生活记录（最近似但我们做的是不同的事）
- **定位**: AI驱动的全能生活记录伴侣
- **价格**: ¥18/月 ¥198/年
- **核心**: AI情绪分析 + 水晶球交互 + 人生之书
- **优势**: AI能力最强，功能最全，App Store编辑推荐
- **弱点**: 太重了，用户打开有心理负担（"又要记一堆"）
- **对我们的启示**: 我们更轻、更聚焦。心光让用户"记录"，我们让用户"行动"

### Day One — 专业日记（全球标杆）
- **定位**: 专业数字日记，苹果设计奖
- **价格**: 免费 / Silver $49.9/年 / Gold $74.99/年
- **核心**: 纯日记 + 今年加了AI（Daily Chat）
- **优势**: 品牌强，功能成熟，隐私保护标杆
- **弱点**: 价格高，中文用户少，偏欧美
- **对我们的启示**: 不是直接竞品，但说明"日记+AI"是大方向

## 三个核心差异化机会

### 1. 「行动引导」是最大的空白
所有竞品都是"被动记录"（用户自己写日记/选情绪），只有光球版是**主动给用户一个治愈小任务**。这是最独特的价值。

### 2. 「呼吸光球」的仪式感
MOODA 的房间装饰、小日常的翻牌反馈、心光的花园…每个产品都有"仪式感时刻"。光球的呼吸动画 + 抽取动作本身就是一个独特的仪式。

### 3. 「极轻」的定位
心光什么都想做（日记+AI+花园+水晶球），用户打开心理负担重。光球版只做一件事：**点一下 → 做一件小事 → 记录情绪**。3秒完成，足够轻。
