## 一、产品定位

一款通过**呼吸光球引导正念 + 随机小事探索**的方式，帮助用户**打破惯性生活、觉察情绪、接纳每一种感受**的情绪记录工具。

**核心理念：** 你不需要好起来，你只是来体验的。

---

## 二、目标用户

- 一二线城市年轻人（20-35 岁）
- 感到生活陷入惯性重复，渴望"活着的感觉"
- 对正念 / 心理学 / 身心灵有了解或好奇
- 不是追求"被治愈"，而是想要"感觉到"

## 四、功能架构

### 4.1 首页 — 呼吸光球
- 暖白色发光球体，`radial-gradient` + 多层 `box-shadow` 实现光晕
- CSS `@keyframes orb-breathe`：12 秒正念呼吸循环
  - 吸气 4s → 光球扩张变亮（scale 0.80 → 1.15）
  - 屏息 4s → 保持充盈状态
  - 呼气 4s → 收缩变暗（scale 1.10 → 0.80）
- 光球下方同步显示文字：吸气… / 屏息… / 呼气…
- 拖拽行为：长按 200ms 后下拉查看时光印记（阈值 20vh）
- 节气日（如立春、夏至等）显示节气卡片替代光球

### 4.2 核心流程

```
首页（光球呼吸）
  │ 点击光球
  ▼
加载动画（700ms）
  │
  ▼
抽取一件小事（按时间段筛选）
  ├─ 再次点击光球 → 无限次换一条
  │
  ├─ 【完成小事】 → 选情绪 → 存为记忆 → 回到首页
  │
  └─ 【记录当下】 → 打开信纸（标题+正文+拍照） → 选情绪 → 存为记忆 → 回到首页
```

### 4.3 小事事件系统
- 50 件治愈小事，按时间段分类（清晨/午间/午后/户外/黄昏/夜晚/随时）
- 抽取时按当前时间段筛选（优先显示适合此时段的事件）
- 关键词标记：晨光、听鸟、观云、拾叶、品茶等

### 4.4 情绪系统
- 7 种情绪：平静、温暖、开心、满足、低落、疲惫、困惑
- 每种情绪有对应颜色和反馈文案
- 情绪选择后 3 秒显示反馈气泡
- 每种情绪都获得同等的接纳，不评判好坏

### 4.5 信纸记录
- 点击"记录当下"打开信纸
- 可填写标题 + 正文 + 拍照
- 右上角 ✕ 按钮可退出
- 提交后进入情绪选择

### 4.6 时光印记（时间轴）
- 全屏纵向时间轴，朋友圈式布局
- 左侧情绪色圆点 + 连接线
- 右侧显示：关键词、正文、心情标签、照片
- **玻璃罐萤火虫**：Canvas 绘制，按月切换，情绪色发光点在罐中浮动
- 支持滑动删除（左滑 55px 后显示删除按钮）
- 按日期分组，显示"今天 / 昨天 / x月x日"

### 4.7 四季轻音乐
- 点右上角音符按钮播放/暂停
- 按季节切换曲库（春/夏/秋/冬各 2 首）
- 音量 0.25，循环播放

### 4.8 首次引导
- 首次打开显示遮罩文案
- "不需要好起来，只需要感觉到"
- 点击"开始"后存入 localStorage，不再显示

## 十、背景与核心理念

### 哲学来源
- **《当下的力量》**：过去和未来都是幻觉，只有当下是真的
- **《与神对话》**：我们不是来修正自己的，是来体验的

| 维度 | 今日拾光 | 主流情绪记录 App |
|---|---|---|
| 核心理念 | 不需要好起来 | 自我提升/变好 |
| 交互方式 | 呼吸光球 + 盲盒抽取 | 每日打卡 + 评分 |
| 情绪态度 | 每种情绪都允许 | 追求积极情绪 |

| 商业目的 | 先体验，再考虑付费 | 先卖会员 |
