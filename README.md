# 拾光日记 — 产品设计文档

## 产品定位

一款通过**呼吸光球引导正念 + 随机小事探索**的方式，帮助用户**打破惯性生活、觉察情绪、接纳每一种感受**的情绪记录工具。

---

## 哲学背景（核心灵感来源）

### 《当下的力量》
- 过去和未来都是大脑对自己的幻觉和欺骗
- 人最重要的能力是**活在当下，存在于此刻**
- 痛苦来源于对过去的执着和对未来的焦虑

### 《与神对话》
- 不需要信仰任何宗教，不需要畏惧任何神
- **我们是神的一部分**
- 来到这个世界不是为了学习和修正什么
- 是为了**体验、经历、认识到自己的本质**
- "你不需要被修复。你从来都没有破碎过。"

### 核心一句话
> 你不需要好起来，你只是来体验的。

---

## 核心理念（文案基调）

1. **打破惯性** — 走出复制粘贴的重复生活，做一些小小的探索
2. **活在当下** — 通过每一件小事，感受此刻的真实
3. **觉察情绪** — 用情绪罐子觉察自己的感受，好好照顾自己
4. **接纳每一种情绪** — 哪怕是消极和低落，也同样是被允许的，值得被珍视的
5. **人生重在体验** — 不是来修行的，是来体验的

---

## 目标用户

- 一二线城市年轻人
- 感到生活陷入惯性重复，渴望"活着的感觉"
- 对正念/心理学/身心灵有了解或好奇
- 不是追求"被治愈"，而是想要"感觉到"

---

## 产品功能设计

### 首页 — 呼吸光球
- 一个发光的球体，12秒呼吸循环：
  - 吸气 4s → 光球扩张变亮
  - 屏息 4s → 保持充盈
  - 呼气 4s → 收缩变暗
- 引导用户跟随呼吸节奏，回到当下
- 光球下方显示当前阶段：「吸气…」「屏息…」「呼气…」
- 点击光球随机抽取一件治愈小事

### 小事任务（情绪探索）
- 每天随机抽取一件小事，例如：
  - "去窗边找一朵白色的花"
  - "抬头看看今天的云，画下它的形状"
  - "剥一个橘子，闻闻手上的味道"
  - "认真搓一个泡泡" 等等
- 小事的特点：简单、不费力、让人回到当下
- 按时间段分类（清晨/午间/傍晚/晚间/任何时间）

### 情绪选择
- 完成任务后，选择此刻的情绪颜色
- 6种情绪：平静、温暖、开心、满足、低落、疲惫
- **每种情绪都被允许，不需要"好起来"**
- 选择后显示情绪反馈语（详见下方文案部分）

### 情绪反馈语
| 情绪 | 反馈语 |
|:---|:---|
| 平静 | 你看。你不需要做什么。就已经在了。 |
| 温暖 | 这也是你的一部分。允许它多待一会儿。 |
| 开心 | 喜悦是你的本质。你只是在记起来。 |
| 满足 | 你不缺什么。你一直都很完整。 |
| 低落 | 你允许低落经过你。你没有变成低落。 |
| 疲惫 | 你的灵魂知道你需要停下来。听它的。 |

### 时光轴
- 以时间为线索记录每一次的情绪时刻
- 左侧情绪色球（半透明光晕感）
- 楷体文字，柔和治愈风格

### 背景音乐
- 正念冥想类轻音乐
- 按四季区分曲库
- 低音量、循环播放，不打扰

## 视觉风格要求

- **字体**：霞鹜文楷（LXGW WenKai），古典文艺手写感
- **色调**：暖白、米色、浅棕为主，莫兰迪色系点缀
- **风格关键词**：东方留白、手写温度、柔和发光、毛玻璃
- **呼吸光球**：暖白光晕，核心发光 + 多层外发光，不刺眼
- **情绪色球色值**：
  - 平静 #7BB8D4
  - 温暖 #F0A88C
  - 开心 #F5C07A
  - 满足 #8FC4A8
  - 低落 #B0A0C0
  - 疲惫 #90A8B8

## 首批引导/点题文案（首次打开app）

打开 app 第一眼展示遮罩文案，点击"开始"后进入主页。文案需要传达以下要素：
- 你不需要被修复
- 你来这里是为了体验
- 过去和未来是幻觉，只有现在是真的
- 每一种情绪都是被允许的

## 小红书推广文案方向

- 标题风格：真实分享感，不要"治愈系"这种俗套词
- 正文结构：从"生活惯性"的感受切入 → 介绍光球和任务 → 情绪反馈 → 理念升华
- 核心理念落在：**你不需要变好，你只是来体验的**
- 配卡建议：5张图（光球 → 抽任务 → 选情绪 → 反馈语 → 时光轴）
