# 🌱 今日拾光 — 功能说明

一个温柔的生活记录应用，专为手机端设计。

---

## 📱 交互流程

### 入口：首页自动推荐一件小事
- 打开页面自动显示一条适合当前时段（晨/午/晚/夜）的"光球小事"
- 例如早晨显示"拉开窗帘让阳光洒进来"，夜晚显示"看看今晚的月亮"

### 两条独立记录路径

#### 路径 A：快速记录（+号按钮）
1. 点击底部 **+** 号
2. 选择 **📷 拍照** 或 **✏️ 文字**
3. **直接记录进时间轴** — 不选情绪、不弹窗，一步到位

#### 路径 B：收藏进记忆（光球专属）
1. 看到光球小事的文字描述后
2. 点击 **「收藏进记忆」** 按钮
3. 弹出 **🎨 情绪颜色选择**（平静/温暖/低落/疲惫）
4. 选择情绪 → 显示温柔回应 → 记录存入时间轴

### 今日记录 — 溪流时间轴
- 页面底部以蜿蜒溪流形式展示**今天的所有记录**
- 每条记录显示：时间、关键词、情绪颜色点
- 点击记录可查看详情气泡（事件全文 + 情绪）

### 再拾一段
- 收藏完成后，可点击"再拾一段"获取新的随机小事

---

## 🗂️ 页面结构

| 区域 | 内容 |
|------|------|
| 顶部 | 干支标题（如"壬寅年 · 春分"）+ "今日拾光" |
| 中间 | 节气卡片（有节气时显示诗句/宜做小事） |
| 内容区 | 当前光球小事的文字描述 |
| 底部 | 今日记录溪流时间轴 + 本月记录总数 |
| 右下角 | **+** 号按钮（拍照/文字） |
| 右上角 | **♪** 音乐开关 |

---

## 🎨 情绪颜色

| 情绪 | 颜色 |
|------|------|
| 平静 | #8CB4C4 青灰色 |
| 温暖 | #D4A4A8 粉色 |
| 低落 | #A4C4B8 墨绿 |
| 疲惫 | #C8C888 黄褐 |

情绪在时间轴上以**有色圆点**标识，点击记录可查看详情。

---

## 🎵 四季音乐

右上角音乐按钮根据当前季节播放不同的轻音乐（Pixabay 免费音源）：

| 季节 | 风格 |
|------|------|
| 春 | 温柔自然 |
| 夏 | 温暖钢琴 |
| 秋 | 安静氛围 |
| 冬 | 清冷静谧 |

---

## 💾 数据存储

所有记录保存在浏览器 **localStorage** 中，刷新不丢失。

每条记录包含：
- 时间 / 日期 / 月份
- 小事 emoji + 关键词 + 全文
- 情绪颜色（仅收藏路径有）
- 温柔回应文案（仅收藏路径有）

---

## ✨ 设计理念

- **极简**：两步完成记录，零门槛
- **双路径**：快速记录 vs 有仪式感的收藏
- **四季流转**：音乐与节气随自然变化
- **温柔交互**：每一个操作都有柔软的反馈
