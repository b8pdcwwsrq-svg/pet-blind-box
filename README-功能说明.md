# 🌱 今日拾光 — 功能说明

一个温柔的日记应用，专为手机端设计。每天随机推荐一件小事，拍照写字收集进时间轴，月底在玻璃罐里看情绪的星星。

---

## 📱 交互流程

### 首页 → 光球
- 首页显示呼吸光球（非节气日）或节气卡片（节气当天）
- 点击光球/卡片随机抽取一件当前时段的温柔小事
- 不满意可以再次点击换一条，不限次数

### 光球抽取后
- 看到小事卡片，下面两个按钮：
  - **标记心情** — 选四种情绪之一（平静/温暖/低落/疲惫），情绪色存入
  - **记录当下** — 打开信封作答区，拍照/写字/两者都做

### 记录当下（首页也有这个按钮）
- 打开一个信封式作答区
- 中间是文字框，左下角是相机图标
- 拍照后照片预览出现在文字框上方
- 拍照和写字可以共存，合为一条记录
- 点"存入时光"后进入情绪选择

### 所有记录都过情绪选择
- 平静（青灰）、温暖（粉）、低落（墨绿）、疲惫（黄褐）
- 选完存入时间轴 + 玻璃罐

---

## 📔 时光印记（下拉或点右上角📔进入）

### 玻璃罐萤火虫收集器
- 当月 30/31 天对应罐子里的星星
- 有情绪记录的日子是彩色萤火虫，带情绪色呼吸闪烁 + 十字光芒
- 没有记录的日子是暗淡小白点
- 点击某颗星星可以跳到当天记录
- 顶部 ← → 箭头切换月份，只有有数据的月份可切

### 朋友圈式时间轴
- 按日期分组，左侧是情绪水彩圆点（情绪色、虚边晕染）
- 纯文本卡片，不分标题正文
- 左上角水彩圆点表达情绪，右下角显示时间
- 支持左滑删除

## 🎨 情绪颜色

| 情绪 | 颜色 |
|------|------|
| 平静 | #8CB4C4 青灰色 |
| 温暖 | #D4A4A8 粉色 |
| 低落 | #A4C4B8 墨绿 |
| 疲惫 | #C8C888 黄褐 |

颜色在玻璃罐里表现为星星的色光和光芒，在时间轴上表现为水彩圆点。

---

## 🎵 四季音乐

右上角音乐按钮根据当前季节播放不同的轻音乐（Pixabay 免费音源）。

---

## 💾 数据存储

所有记录保存在浏览器 localStorage 中，刷新不丢失。

每条记录包含：
- 日期 / 时间 / 月份
- 小事文字全文
- 情绪颜色
- 照片（可选）

---

## 🗂️ 页面结构

| 区域 | 内容 |
|------|------|
| 顶部 | 干支标题（如"丙午年 · 夏至"）+ "今日拾光" |
| 中间 | 节气卡片（节气日）或呼吸光球（非节气日） |
| 内容区 | 当前光球小事的文字描述 |
| 下方 | 标记心情 / 记录当下 / 换一条 |
| 底部 | 记录当下按钮（首页常驻） |
| 右上角 | 📔 时光印记入口 / ♪ 音乐开关 |

---

## ✨ 设计理念

- 温柔——没有催促，没有打卡，没有连击
- 私密——是自己的日记本，不是社交平台
- 探索——光球随机抽取，换上新的，选中就是今天的心情
- 视觉统一——莫兰迪配色 + 水彩晕染 + 纸张纹理 + 玻璃罐星星
